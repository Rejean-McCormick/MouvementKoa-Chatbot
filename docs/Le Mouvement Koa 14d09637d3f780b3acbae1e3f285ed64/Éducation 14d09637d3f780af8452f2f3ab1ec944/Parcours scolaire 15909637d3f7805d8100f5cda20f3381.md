# Parcours scolaire

Pour chaque âge cible, il y a 7 niveaux de compétence: Initié, Apprenti, Praticien, Confirmé, Avancé, Expert, Maître. L’objectif est atteint avec le niveau “Confirmé”.
Il y a 8 Matières (domaines d’étude).
